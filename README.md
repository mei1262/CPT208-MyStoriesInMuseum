# MyStoryInMuseum - CPT208 A3 项目

**让每一次参展都成为你的故事**  
XR + AI 虚拟导览 · 博物馆个性化陪伴平台

**Live Demo**: https://luck559.github.io/CPT208-MyStoriesInMuseum/

## 🎮 3 个 Must-Have Playful Features
1. **个性化 AR 虚拟伙伴陪伴** —— 自定义形象 + 语音 + 实时 AR 跟随讲解  
2. **AI 共创纪念视频** —— 上传照片后 AI 自动生成专属参展故事视频  
3. **展区打卡 + 故事碎片收集** —— 专为缓解长时间 XR 佩戴疲劳而设计的展区切换机制

## 📁 项目结构
- `index.html` —— 主系统入口（三大角色模块）
- `ar-demo.html` —— AR 实时演示页（核心功能展示）
- `assets/` —— 图片与 3D 模型
- `ai-logs/` —— AI 使用记录（必须保留）
- `PosterSources/` —— 海报源文件

**技术栈**：HTML + Tailwind CSS + Vanilla JS + Google model-viewer (AR)

**AI 使用**：所有 AI 生成代码与提示词均记录在 `/ai-logs/` 文件夹中。
